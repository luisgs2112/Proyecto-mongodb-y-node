 Sistema de Gestión Educativa - MongoDB & Node.js

Este proyecto es una aplicación de consola desarrollada en Node.js que interactúa con una base de datos NoSQL (MongoDB) para gestionar un sistema de cursos online.

1. Requisitos del Proyecto
- 5 Colecciones: Estudiantes, Cursos, Instructores, Pagos y Categorías.
- Campos: Cada documento cuenta con al menos 4 campos descriptivos.
- Módulo CRUD: Implementación de Crear, Leer, Actualizar y Borrar.
- Consulta Sencilla: Filtro y conteo de documentos en tiempo real.

2. Tecnologías Utilizadas
- Node.js
- MongoDB (Local/Compass)
- Mongoose (ODM)
- Dotenv (Variables de entorno)
